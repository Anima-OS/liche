`liche` is a command line tool to check links in Markdown and HTML files.
It checks all `a` and `img` tags in specified files.

## Installation

```sh
go get -u github.com/raviqqe/liche
```

## Usage

```sh
liche file.md
liche file1.md file2.md
```
