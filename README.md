`linkcheck` is a command line tool to check links in Markdown and HTML files.
It checks all `a` and `img` tags in specified files.

## Installation

```sh
go get -u github.com/raviqqe/linkcheck
```

## Usage

```sh
linkcheck file.md
linkcheck file1.md file2.md
```
